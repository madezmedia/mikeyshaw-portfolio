## 🚀 Quick Start

### Prerequisites
- Node.js 18+
- npm or pnpm

### Installation
```bash
# Clone the repository
git clone https://github.com/madezmedia/mikeyshaw-portfolio.git

# Navigate to project directory
cd mikeyshaw-portfolio

# Install dependencies
pnpm install

# Run development server
pnpm run dev
```
